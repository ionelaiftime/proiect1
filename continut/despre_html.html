Limbajul Html

HTML(HyperText Markup Language) este un limbaj de marcare utilizat pentru a crea pagini web.

Un exemplu de cod html îl gasiți în imaginea urmatoare care se afla la adresa https://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png.

Imaginea are atribut height de valoare 147.

Tag-urile html cele mai utilizate în realizarea tabelelor sunt:

table definește un tabel
tr un rând dintr-un tabel
td o celula dintr-un tabel

Observație: Pentru a pune contur unui tabel se poate(dar nu se recomandă) să se pună atributul border="1".

La validare va da eroarea: The border attribute on the table element is obsolete.

Formularele se pot crea cu ajutorul tag-ului form care poate conține tag-urile: input, testarea, label, select, option, button.

Pentru a crea liste se pot folosi tag-urile:

  1. ul- listă neordonată
  2. ol- listă ordonată
  3. li- item din listă
  4. dl- listă de descrieri
  5. dt- termen dintr-o listă cu descrieri
  6. dd- descriere dintr-o listă cu descrieri

Pentru a insera un document HTML în altul se poate folosi tag-ul <iframe>.

Astfel, exemplul de mai jos insereaza conținutul de la adresa https://html.spec.whatwg.org/multipage/.
